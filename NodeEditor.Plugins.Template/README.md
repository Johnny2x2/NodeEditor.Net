# Plugin Template (SDK)

Copy this folder to create a new plugin with a single sample node.

## Quick Steps

1. Duplicate the folder and rename it (project + namespace).
2. Update the project file name and root namespace (if desired).
3. Edit plugin.json with a new id, name, entryAssembly, and category.
4. Update TemplatePlugin.cs to your plugin class and nodes.
5. Add or update plugin-marketplace.json (optional but recommended).
6. Build + package with publish-plugin.ps1.
